wangEditor demo basic

以上demo是wangEditor的最基本应用。引用css和js文件，为textarea执行一个wangEditor()方法，即可生成富文本框。

第一步，引用wangEditor的css文件，css文件最好在<head>中引用

<link rel="stylesheet" type="text/css" href="css/wangEditor-1.3.css">
<!--[if IE]>
<link rel="stylesheet" type="text/css" href="css/wangEditorFont-ie7-1.3.css"> <--兼容IE7-->
<![endif]-->

第二步，引用jquery和wangEditor.js，js文件最好在<body>最下方引用

<script type="text/javascript" src='js/jquery-1.10.2.min.js'></script>
<script type="text/javascript" src='js/wangEditor-1.3.js'></script>

第三步，配置富文本框。想要获取富文本框的html代码，直接获取textarea的值即可。

<script type="text/javascript">
    $(function(){
        $('#textarea1').wangEditor();
    });
</script>

如果想要在一个页面创建多个富文本编辑器，也很简单。可参见 demo-multi-text.html

<!--先在html中定义两个textarea，然后：-->
<script type="text/javascript">
    $(function(){
        $('#textarea1').wangEditor();
        $('#textarea2').wangEditor();
    });
</script>

最后，除了以上基本应用之外，wangEditor还支持多种配置，具体的参见相应的demo页面说明。

$('#textarea1').wangEditor({
	'$initContent': ...    //配置要初始化的内容
	'menuConfig': ...      //配置要显示的菜单（其他的隐藏）
	'onchange': ...        //配置onchange事件
});
